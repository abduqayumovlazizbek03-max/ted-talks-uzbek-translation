# TED Talks Uzbek Translation Project

This repository contains student-led translations of selected TED Talks
from English to Uzbek.

## Project Scope
- Translation (EN → UZ)
- Subtitle editing and synchronization
- Voice-over script preparation
- Audio recording and basic editing

## Purpose
To make global educational content accessible to Uzbek-speaking audiences.

## Tools Used
- Manual translation
- Subtitle editing
- Basic audio tools

## Disclaimer
This is a non-commercial, educational project.
All original content belongs to TED.
